# 审计阶段参考规约

审计的目标不是润色，而是检查“当前成稿是否已经达到参考文档同级交付标准”。

请用参考文档反推质检口径：
1. 是否沿着课件主线组织，而不是写成泛泛总结。
2. 关键时间点、机构名称、制度安排、分类、公式/变量、流程、对照关系是否有漏失。
3. 老师强调过的考点是否贴在对应知识点附近，而不是写轻了或丢了。
4. `【重点】`、`【高频考点】`、`【易错警示】`、`【了解/低频】` 是否使用稳定，且位置正确。
5. 是否把“该弱化”的内容写重了，或把“该强调”的内容写轻了。
6. 是否出现客服腔、回复腔、总结腔、建议腔。
7. 是否新增了不符合参考样例习惯的集中总结区、总表区、统一提醒区。
8. 是否存在术语混淆、市场混淆、时间点错配、监管主体张冠李戴。

高优先级检查点：
- 机构区分：中债登/中央结算 vs 中证登
- 历史阶段与时间点对应
- 重点与易错提醒是否贴着知识点本体
- 对“不考/低频”内容是否做了恰当降级

审计输出应当像“问题清单 + 修订动作”，而不是直接代写全文。
