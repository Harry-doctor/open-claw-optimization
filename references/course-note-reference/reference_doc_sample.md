中国的金融体系与债券市场
说明：本版已完成去重、结构统一与表述收束。保留“【重点】/【高频考点】/【易错警示】/【了解/低频】”标签体系，正文以考试复习可直接使用为目标。

## 一、债券托管、登记与结算

### 1. 银行间市场与托管结算机构
银行间债券市场的托管结算，主要由中央国债登记结算有限责任公司承担，课堂中常简称为中央结算或中债登。
【重点】做题时要把“中央结算/中债登”与“中证登”区分开。
【高频考点】不同市场对应的托管、登记体系不同，不能混记。

### 2. 国债柜台市场的二级托管制度
国债柜台市场实行二级托管制度。
流程式抓手：
投资者 → 商业银行（二级托管）→ 中央结算公司（一级托管）
商业银行既承担经销角色，也承担投资者接触层面的托管服务。
【重点】二级托管的“二级”说的是托管层级，不是两个市场。
【高频考点】银行间市场与国债柜台市场在托管安排上的差异。
【易错警示】不要把国债柜台市场的托管规则误认为交易所场内规则。

### 3. 债券登记的基本含义与类型
债券登记可以理解为：记录债券持有人是谁，以及持有人发生变化时如何更新。
主要包括三类：
- 初始登记：债券发行后，首批持有人登记在册。
- 变更登记：债券买卖转让后，持有人信息变更。
- 退出登记：债券到期或退出后，持有人登记关系注销。
流程式抓手：
初始登记 → 变更登记 → 退出登记
【重点】三类登记的定义和作用要对应起来。
【高频考点】登记在债券发行、流转、退出中的功能。
【易错警示】“托管”“登记”“结算”不是完全同义概念，做题时先分清题干在考哪一层。

### 4. 场内与场外市场的登记体系差异
场外市场（如银行间市场）最终登记一般在中债登/中央结算公司。
场内市场（上交所、深交所）股票、债券、基金等的登记体系，通常对应中国证券登记结算有限责任公司（中证登）。
【重点】先判断题目讲的是场内还是场外，再判断对应登记安排。
【高频考点】中债登与中证登的对应关系。

## 二、中国金融体系的历史演进

### 1. 1949—1978年：计划经济时期的大一统金融体系
#### （1）统一人民币制度
1948年12月1日，中国人民银行正式成立。
第一套人民币同时发行。
【重点】1948年中国人民银行成立，是时间点题中的常见考点。
【高频考点】不要机械记成“新中国成立后才有中国人民银行”，两者时间并不完全重合。

#### （2）“大一统”金融体系的含义
所谓“大一统”，核心是当时金融体系高度集中统一管理。
其主要表现为：
- 中国人民银行分支机构覆盖全国；
- 中国人民银行是该时期唯一的银行；
- 中国人民银行同时承担中央银行职能与商业银行职能；
- 全国实行统一的信贷资金管理体制。
可理解为：中国人民银行在当时集现金中心、结算中心、信贷中心等职能于一体。
【重点】“唯一银行 + 双重职能”是理解大一统金融体系的抓手。
【高频考点】考试喜欢问“大一统”具体指什么。
【易错警示】题目若把“大一统”说成多银行并存、分工明确，通常是错项。

#### （3）统存统贷
在该时期，全国实行统存统贷的信贷资金管理体制。
【重点】“统存统贷”是规范表述。
【易错警示】不要把它误记成自造词，如“存贷挂钩”等。

### 2. 1979—1983年：多元混合型金融体系
改革开放初期，金融体系开始由高度统一走向分化。
这一阶段的代表变化包括：
- 恢复设立中国农业银行；
- 中国银行从中国人民银行分设出来；
- 中国建设银行从财政部分离出来；
- 中国信托投资公司、中国人民保险公司等非银行金融机构先后成立。
流程式抓手：
统一管理 → 职能分化 → 多元混合
【重点】抓住“分设”“分离”“非银行金融机构成立”三类变化。
【高频考点】改革开放后金融体系开始多元化，不再是原来的绝对大一统状态。

### 3. 1984—1993年：金融体系逐步成形，资本市场初步建立
#### （1）金融体系逐步成形
这一时期形成了以中央银行为中心、国家专业银行为主体、多种金融机构分工协作的金融体系。
【了解/低频】这一表述更偏整体概括，做了解即可，但题目出现时要能识别对应历史阶段。

#### （2）资本市场初步建立的关键时间点
时间轴式抓手：
- 1990年12月：上海证券交易所正式开业；
- 1991年7月：深圳证券交易所正式开业；
- 1992年10月：国务院证券委员会和中国证券监督管理委员会成立，中国资本市场开始逐步纳入统一监管框架。
【重点】上交所、深交所、证监会相关时间点是高频记忆点。
【高频考点】90年12月上交所、91年7月深交所、92年10月监管框架建立，容易连着考。

## 三、多层次资本市场与股票市场发展

### 1. 股票市场发展的早期脉络
课堂回顾了中国股票市场早期发展，提到飞乐音响作为公开发行股票的早期代表之一。
证券市场早期探索中，还经历了企业债转让、国库券柜台转让、证券公司设立等环节。
【了解/低频】早期探索性事件更适合作为历史背景理解。

### 2. 股权分置改革
股权分置改革的核心，是推动原有非流通股逐步转为可流通状态，实现更完整的市场流通机制。
【重点】理解“股权分置改革”的本质，不要只背名字。
【高频考点】题目如果考改革意义，通常围绕“流通机制完善”“市场化程度提升”展开。
老师补充：市场下跌不能总是机械归因于单一外部事件，股改后部分解禁带来的供给冲击也会影响市场表现。
【易错警示】做题时不要只记一个孤立原因，要注意供给侧变化的影响。

### 3. 多层次资本市场的基本结构
课堂提到的主要板块包括：主板、创业板、科创板、北交所、全国股转系统（新三板）等。
北交所与全国股转系统之间具有衔接关系，可理解为中小企业分层培育、逐步升级的通道。
【重点】多层次资本市场考查时，通常围绕板块定位、服务对象、准入门槛来出题。
【高频考点】不要把已调整的历史板块口径按旧版本死记。
【易错警示】中小板已并入深市主板，现行记忆中不要再把中小板当作独立板块。

## 四、外汇、黄金与期货衍生品市场

### 1. 外汇市场
银行间外汇市场的交易方式包括：竞价交易与询价交易。
课堂强调：在实际理解中，要记住询价交易占重要地位。
【了解/低频】外汇市场交易方式常作识别型考点。

### 2. 黄金市场
商业银行柜台可办理黄金相关业务。
纸黄金更适合理解为一种账户化、凭证化的黄金投资方式，而不是直接持有实物黄金。
【易错警示】不要把纸黄金和实物黄金投资混成一回事。

### 3. 期货与衍生品市场
期货与衍生品市场诞生的原始目的，是规避价格波动风险。
课堂用商品价格波动举例，帮助理解套期保值的逻辑。
流程式抓手：
现货价格波动风险 → 使用期货/衍生品对冲 → 降低不确定性影响
【重点】衍生品最初不是为了“投机”而产生，而是为了风险管理。
【高频考点】“规避风险/套期保值”是期货市场基础定位。

### 4. 重要时间点
时间轴式抓手：
- 1990年10月：郑州商品交易所成立；
- 2010年4月16日：沪深300股指期货推出；
- 2015年2月：上证50ETF期权上市。
【高频考点】时间点和交易场所对应关系容易混考。
【易错警示】上证50ETF期权在上海证券交易所上市，不是中国金融期货交易所。

## 五、影响金融市场的经济因素与宏观政策

### 1. 经济因素的基本框架
课程进入“影响金融市场的因素”模块后，老师提醒要区分：经济因素、技术因素、心理因素。
题目喜欢把不同模块内容混在一起考。
【高频考点】遇到综合题时，先判断它考的是“经济、技术、心理”中的哪一类。

### 2. 经济增长与 GDP
课堂强调，现在更常用 GDP 观察经济运行。
GDP 按境内地理范围统计，不以国籍区分，这一点要和 GNP 区分。
【重点】GDP/GNP 区分属于基础但容易混淆的考点。

### 3. 经济周期
流程式抓手：
复苏 → 繁荣 → 衰退 → 萧条
【高频考点】顺序型记忆必须记牢，不能前后颠倒。

### 4. 通货膨胀、利率与金融市场
当通货膨胀低于预期时，利率可能下降，资金更愿意流向消费和投资，债券、股票价格往往得到支撑。
当通货膨胀高于预期时，利率可能上升，市场流动性趋紧，债券、股票价格承压。
【重点】这一部分按理论框架记忆即可。
【易错警示】理论关系可以用于做题，但现实中不要机械绝对化理解。

### 5. 宏观经济政策
宏观经济政策主要包括：货币政策、财政政策、收入政策。
关系式抓手：
收入政策（更高层约束与引导） → 影响财政政策 / 货币政策作用方向与力度
【高频考点】没有单独的“支出政策”这一分类口径，题目中容易拿来混淆。

### 6. 收入政策：总量目标与结构目标
收入政策可以分为：
- 总量目标：着眼宏观总量平衡，如积累与消费、失业与通胀等关系；
- 结构目标：着眼不同收入、不同支出之间的比例关系。
老师用“主业 + 兼职”举例，说明结构问题的本质在于占比与配置，而不只是有没有收入。
【重点】结构问题的抓手是“比例”。
【高频考点】题目一旦出现“比例”这个关键词，通常更偏向结构目标。

### 7. 菲利普斯曲线
课堂用菲利普斯曲线解释就业与通货膨胀之间的关系：一般而言，就业率越高，通胀压力越大；就业率越低，通胀压力越小，甚至可能出现通缩倾向。
关系式抓手：
就业率上升 ↔ 通胀压力上升
就业率下降 ↔ 通胀压力下降 / 通缩风险上升
【重点】菲利普斯曲线体现了就业与通胀之间的权衡关系。
【高频考点】这部分常与宏观调控、总量目标、收入政策一起综合考查。
对政策层面的理解是：政府调控更多是在可承受区间内做平衡，而不是同时无成本地实现所有理想目标。
