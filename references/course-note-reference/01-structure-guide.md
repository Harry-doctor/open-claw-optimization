# 结构提纯阶段参考规约

这一阶段继承的是“参考文档的组织方法”，不是照抄句子。

必须遵守：
1. 以课件主线/知识点主线展开，不写成松散摘要。
2. 优先保留：定义、分类、时间点、机构名称、制度安排、流程、对照关系、老师明确强调的考点。
3. 老师补充必须贴着对应知识点放置，不要单独散落成聊天记录式备注。
4. 标签体系只保留并内嵌在对应知识点附近：`【重点】`、`【高频考点】`、`【易错警示】`、`【了解/低频】`。
5. 允许使用“流程式抓手 / 时间轴式抓手 / 关系式抓手 / 对照式抓手”帮助后续成稿，但前提是服务知识点理解，不是堆形式。
6. 对老师明确说“不考 / 基本不考 / 不需要记 / 简单了解”的内容，降级处理即可，不要完全抹掉。
7. 不要额外产出全文总结、集中考点汇总、客服式说明。
8. 不确定术语、OCR错字、转写疑点继续用 `【待核】` 标记。

从参考文档中抽出的结构特征：
- 机构/市场类知识点：先讲对象，再讲制度，再给易错区分。
- 历史演进类知识点：优先转成时间轴，再嵌重点和高频考点。
- 宏观政策/理论类知识点：优先转成关系式或对照式抓手，避免空泛叙述。
- 重点提示必须贴着知识点本体，而不是另起“本章重点”。

代表片段（只学结构，不要照抄）：

## 片段A：制度 + 易错区分
1. 银行间市场与托管结算机构
银行间债券市场的托管结算，主要由中央国债登记结算有限责任公司承担。
【重点】做题时要把“中央结算/中债登”与“中证登”区分开。
【高频考点】不同市场对应的托管、登记体系不同，不能混记。

## 片段B：时间轴组织
时间轴式抓手：
1990年12月：上海证券交易所正式开业；
1991年7月：深圳证券交易所正式开业；
1992年10月：国务院证券委员会和中国证券监督管理委员会成立。
【重点】上交所、深交所、证监会相关时间点是高频记忆点。

## 片段C：对照/关系式组织
收入政策可以分为：
总量目标：着眼宏观总量平衡；
结构目标：着眼不同收入、不同支出之间的比例关系。
【重点】结构问题的抓手是“比例”。
【高频考点】题目一旦出现“比例”这个关键词，通常更偏向结构目标。
